# Lightweight-back-office

This project is a lightweight back office to manage a dataset of question.
It run with Vuejs 3.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
